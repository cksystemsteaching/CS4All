CS4All

This is the homepage of the computer science for all course by Professor Christoph Kirsch.

View on GitHub Download .zip Download .tar.gz

This course motivates and explains the basic principles of computer science. We will introduce all relevant layers and study their connection, from the lowest levels of machine architecture via programming languages, algorithms, compilers, and virtual machines to the highest levels of cloud computing. For this purpose we use code examples from the Selfie Project which is an educational system of a self-compiling C compiler, a self-executing MIPS emulator, and a self-hosting MIPS hypervisor. Discussions are held in the #cs4all slack channel.

Syllabus

  1. Introduction
  2. Architecture
  3. Programming Languages
  4. Algorithms
  5. Complexity
  6. Computability
  7. Compilers
  8. Data Structures
  9. Memory Management
  10. Concurrency
  11. Virtual Machines
  12. Cloud Computing
